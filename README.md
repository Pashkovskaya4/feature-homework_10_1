# Модули с функциями для работы с банковскими картами и счетами
## Модуль "widget" показывает информацию о карте/счете, дату и время
## Модуль "masks" маскирует номер карты и счета
## Модуль "processing" сортирует операции по дате и по параметру "выполнено/отказано"
## Модуль "generators.py" сортирует по заданной валюте транзакции,возвращает описание транзакции,генерирует номера банковских карт.
## Модуль "decorators.py" логирует результат и время выполнения функции в файл,либо в консоль.
# Тестирование
## Проект покрыт тестами с использованием библиотеки pytest. Тесты проверяют основную функциональность

# Запуск тестов

# Установить зависимости для тестирования
pip install pytest pytest-cov

# Запустить все тесты
pytest
